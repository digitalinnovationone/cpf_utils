# CpfUtils

Esta gem serve para fazer validação e formatação de CPF, utilize e seja feliz :)

## Instação

Adicionar em seu Gemfile

```ruby
gem 'cpf_utils'
```

Para instalar rode o comando abaixo
```shell
bundle install
```

Ou instalar diretamente sem um Gemfile:
```shell
gem install cpf_utils
```
